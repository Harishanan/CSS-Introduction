CSS selectors

CSS helps to attract the webpage. Making an external style sheet saves a lot work. CSS can be represent using following format,

id     {property     value}
h1 {color:blue;font-size:12px;}
Visit to CSS

  1. Element selector
    • It selects a particular element and styles it up.
    • For, examaple selecting the <h2> element and styling it up using the properties and values, will make all h2 in same format.
  2. ID selector
    • It selects a particular single element and styles it.
    • For this, we use id property and assign value with "" in html.
    • In CSS, we denote this using # symbol.
  3. Class selector
    • It styles group of any elements. They can be a heading or paragraph.
    • Here, have to use . before the name in CSS and the id in html.
  4. Psuedo selector
    • It defines special state of elements.
    • For example: styling an element when user mouses over it, styling visited and unvisted links differently and styling an element when it gets focus.
